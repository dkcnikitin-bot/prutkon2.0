# 🚀 Новые функции импорта v18.25.3

## ✅ Что добавлено

### 1. **Массовое удаление товаров** 🔴

**Где:** Кнопка в хедере (появляется при выборе товаров)

**Как работает:**
- Отметьте чекбоксами нужные товары в таблице
- Кнопка **"Удалить выбранные (N)"** появится автоматически
- Нажмите → подтвердите → товары удалены

**Особенности:**
- ✅ Счётчик показывает количество выбранных
- ✅ Подтверждение перед удалением
- ✅ Сохранение в локальное хранилище и Firebase
- ✅ Уведомление о результате

---

### 2. **Умное определение категории по названию листа** 📂

**Где:** Шаг 1 импорта Excel → выбор категории

**Как работает:**
Система анализирует название листа Excel и автоматически выбирает категорию:

| Название листа | Категория |
|----------------|-----------|
| `Пальцы`, `Paltsy` | Пальцы |
| `Заготовки`, `Zagotovki` | Заготовки |
| `Транспортеры`, `Transportery` | Транспортеры |
| `Ролики`, `Roliki` | Ролики |
| `Ремни`, `Remni` | Ремни |
| `Метизы`, `Metizy` | Метизы |
| `Прутки`, `Prutki` | Прутки |
| `Замки`, `Zamki` | Замки |

**Алгоритм:**
1. Прямое совпадение (100%)
2. Содержит название категории (80%)
3. Содержит синоним (70-75%)

**Пример:**
```
Лист Excel: "Пальцы конвейерные"
→ Найдено совпадение: "Пальцы" (80%)
→ Категория выбрана автоматически ✅
```

---

### 3. **Умное определение полей из схемы категории** 📋

**Где:** Шаг 2 импорта → таблица маппинга

**Как работает:**
Система загружает схему выбранной категории и добавляет её поля в выпадашку:

**Пример для категории "Пальцы":**
```javascript
schema: ['tech_type', 'available', 'drawing', 'art_prutkon']
```

**В выпадашке появятся:**
- ⚙ Тип техники
- ✅ Доступен для заказа
- 📐 Чертёж
- 🏷 Артикул Пруткон

**Преимущества:**
- ✅ Не нужно создавать динамические поля вручную
- ✅ Все поля категории доступны сразу
- ✅ Автоопределение работает для полей схемы

## 🎯 Полный рабочий процесс

### Импорт с новыми функциями:

**1. Загрузка файла**
```
→ Загрузили Excel
→ Выбран лист "Пальцы"
→ 🔥 Категория определена автоматически: "Пальцы"
```

**2. Маппинг полей**
```
→ Выбрана категория "Пальцы"
→ 🔥 Загружена схема категории
→ 🔥 Добавлены поля: tech_type, available, drawing...
→ Автоопределение: Photo_Path, Photo_Filename найдены
```

**3. Импорт**
```
→ Запустили импорт
→ Все поля сохранены
→ Фото подгружены из папки
```

**4. Очистка**
```
→ Нашли дубли/мусор
→ Отметили чекбоксами
→ 🔥 Массовое удаление (N)
```

## 💡 Советы

### Для лучшего автоопределения категории:
- Называйте листы точно по категориям: `Пальцы`, `Заготовки`
- Или используйте синонимы: `Paltsy`, `Zagotovki`

### Для работы с полями категории:
- Настройте схему категории заранее (через управление категориями)
- Система автоматически подхватит все поля

### Для массового удаления:
- Используйте поиск/фильтры для быстрого отбора
- Отметьте всё чекбокс в заголовке таблицы
- Проверьте перед удалением!
