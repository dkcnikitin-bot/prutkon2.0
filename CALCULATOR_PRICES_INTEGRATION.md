# 🧮 Калькулятор + Прайсы — Объединение (v18.25.4)

## Новые возможности

### 1. Улучшенный поиск клиентов в Битрикс24

**Что исправлено**:
- ✅ Добавлена визуальная индикация поиска (вращающийся спиннер)
- ✅ Автоматический поиск после ввода 2+ символов с задержкой 500мс
- ✅ Увеличенные и более заметные результаты поиска
- ✅ Подсветка при наведении на результаты
- ✅ Уведомление о количестве найденных контактов
- ✅ Кнопка быстрого создания клиента если не найден

**Как использовать**:
1. Начните вводить имя или телефон в поле "Поиск по CRM"
2. Через 500мс автоматически запустится поиск
3. Появится список с индикатором загрузки
4. Нажмите на контакт чтобы выбрать
5. Если не найден — нажмите "КЛИЕНТ НЕ НАЙДЕН" чтобы создать

### 2. Объединение калькулятора с прайсами

**Новый блок "ВЫБОР ТОВАРА ИЗ БАЗЫ"**:

#### Шаг 1: Выберите категорию
Выпадающий список показывает все категории из прайсов:
- Пальцы
- Заготовки
- Транспортеры
- Ролики
- Ремни
- и т.д.

#### Шаг 2: Выберите тип товара
После выбора категории автоматически подгрузятся типы:
- Например: "Палец", "Втулка", "Ось"
- Или "Все типы" чтобы увидеть все товары

#### Шаг 3: Выберите конкретный товар
Список показывает:
- Артикул
- Наименование
- Цену
- Остаток на складе (✅ есть / ❌ нет)

Максимум 100 товаров в списке (сортируются по артикулу).

#### Шаг 4: Автоматическое заполнение
При выборе товара:
- ✅ Артикул подставляется в поле "Артикул транспортера"
- ✅ Наименование подставляется в поле "Наименование"
- ✅ Цена отображается в блоке информации
- ✅ Остаток на складе показывается цветом

## 📋 Техническая информация

### Новые функции в `calculator.js`

```javascript
// Загрузка всех товаров из всех прайсов
window.loadAllProducts()

// Заполнение категорий
window.populateProductCategories()

// Загрузка типов для категории
window.loadProductTypes()

// Загрузка товаров по типу
window.loadProductsByType()

// Выбор товара и автозаполнение
window.selectProductFromBase()
```

### Структура данных

```javascript
window.allProducts = [
    {
        id: 12345,
        art: "001-RTI-25",
        name: "Палец 12x50",
        price: 150.50,
        stock: 100,
        category: "Пальцы",
        source: "prices"  // или "trans"
    },
    // ...
]
```

### Объединённые источники

| Источник | Переменная | Кол-во товаров |
|----------|-----------|----------------|
| Основной прайс | `window.dbProducts` | X товаров |
| Транспортеры | `window.dbTransProducts` | Y товаров |
| **Итого** | `window.allProducts` | **X+Y товаров** |

## 🧪 Тестирование

### Тест 1: Поиск клиента
1. Откройте калькулятор
2. Введите "Иван" в поле поиска
3. Через 1-2 сек должен появиться спиннер
4. Должен показаться список контактов
5. Нажмите на контакт — он должен подписаться

### Тест 2: Выбор товара из прайса
1. В блоке "ВЫБОР ТОВАРА ИЗ БАЗЫ" выберите категорию "Пальцы"
2. В поле "Тип" выберите "Палец" или "Все типы"
3. В поле "Выберите товар" найдите нужный
4. Товар должен автоматически подписаться в форму
5. Информация о товаре должна появиться в зеленом блоке

### Тест 3: Создание заказа с выбранным товаром
1. Выберите товар из базы
2. Укажите количество
3. Нажмите "В корзину"
4. Товар должен добавиться в корзину с правильной ценой
5. Оформите заказ — данные должны сохраниться
