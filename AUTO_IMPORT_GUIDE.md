### 1. **Автоматическое распознавание колонок**
Система **автоматически** определяет поля для фото по:

| Название колонки в Excel | Распознаётся как |
|--------------------------|------------------|
| `Photo_Path` | 📁 Путь к фото |
| `Photo_Filename` | 🖼 Имя файла |
| `Путь к фото` | 📁 Путь к фото |
| `Фото файл` | 🖼 Имя файла |
| `C:\...\extracted_xlsx\...` | 📁 Путь к фото (по данным) |
| `row_0002_img_01.png` | 🖼 Имя файла (по расширению) |

### 2. **Обработка абсолютных путей**
```
В Excel: C:\Users\Никитин Иван\Documents\Пруткон 3\extracted_xlsx\1\3\Paltsy\
В базе:  extracted_xlsx/1/3/Paltsy/
```

### 3. **Автоматическая сборка пути**
```javascript
// Если указаны оба поля:

// Результат: extracted_xlsx/1/3/Paltsy/row_0002_img_01.png
```

## 🎯 Как работает автоимпорт

### Шаг 1: Загрузка Excel
1. Откройте **Прайс (Заготовки)**
2. Нажмите **«Импорт Excel»**
3. Загрузите файл

### Шаг 2: Автоопределение (НОВОЕ!)
Система **автоматически**:
- ✅ Распознаёт колонку `Photo_Path` по названию
- ✅ Распознаёт колонку `Photo_Filename` по названию
- ✅ Анализирует данные (ищет `extracted_xlsx`, `.png`, `.jpg`)
- ✅ Определяет абсолютные пути Windows

### Шаг 3: Проверка
В таблице маппинга вы увидите:
- 📁 Для путей: `📁 extracted_xlsx/1/3/Paltsy`
- 🖼 Для файлов: `🖼 row_0002_img_01.png` (с превью)

### Шаг 4: Импорт
Нажмите **«Далее»** → **«Запустить импорт»**

## ⚠️ Возможные проблемы

### 1. Фото не распознаются
**Причина:** Колонки называются нестандартно  
**Решение:** Назначьте поля вручную в шаге 2:
- Найдите колонку с путями → выберите `📁 ФОТО — папка`
- Найдите колонку с именами → выберите `🖼 ФОТО — имя файла`

### 2. Фото не отображаются после импорта
**Причина:** Файлы не извлечены из Excel  
**Решение:** См. `EXTRACT_IMAGES_GUIDE.md`

### 3. Неправильный путь
**Причина:** В Excel указан путь к другой папке  
**Решение:** Исправьте пути в Excel через Найти-Заменить (Ctrl+H):
- Найти: `C:\Users\OldPath\`
- Заменить: `C:\Users\NewPath\`

## 🎁 Бонусы

### Умное превью
Если файл изображения существует, в таблице показывается:
- 🖼 Миниатюра изображения
- 📝 Имя файла (если изображение не найдено)

### Логирование
Первые 5 товаров с фото выводятся в консоль для проверки.

### Приоритет фото
Поля `photo_filename` и `photo_path` имеют **высший приоритет** и не попадут в другие поля.

4. Запустите `test_auto_mapping.html` для проверки логики
