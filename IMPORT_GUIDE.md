# 📘 Руководство по импорту из Excel в ПРУТКОН ERP

## 🚀 Быстрый старт

### Шаг 1: Подготовка Excel-файла

**Требуемые колонки:**
- **Артикул** (обязательно) — уникальный идентификатор товара
- **Наименование** — название товара
- **Цена** — стоимость в рублях
- **Остаток** — количество на складе

**Для загрузки фото (опционально):**
- **Photo_Filename** — имя файла фотографии (например: `12345.jpg`)
- **Photo_Path** — путь к папке с фото (например: `extracted_xlsx/Paltsyi`)

### ⚠️ Важное замечание об абсолютных путях

Если в Excel указаны **абсолютные пути** (например: `C:\Users\...\extracted_xlsx\1\3\Paltsy\`):
- ✅ Система **автоматически обрежет** путь до `extracted_xlsx/...`
- ✅ Изображения должны существовать по относительному пути

**Пример обработки:**
| В Excel | После обработки |
|---------|-----------------|
| `C:\Users\Name\Docs\extracted_xlsx\1\3\Paltsy\` | `extracted_xlsx/1/3/Paltsy` |
| `row_0002_img_01.png` | `row_0002_img_01.png` |
| **Итог:** | `extracted_xlsx/1/3/Paltsy/row_0002_img_01.png` |

### Как извлечь изображения из Excel

Изображения хранятся внутри Excel файла в папке `xl/media/`. 

**Способ 1: Вручную**
1. Переименуйте `.xlsx` файл в `.zip`
2. Откройте архив
3. Извлеките файлы из `xl/media/`
4. Поместите в соответствующие папки `extracted_xlsx/...`

**Способ 2: Через найти-заменить в Excel**
Если изображения уже извлечены:
1. Откройте Excel
2. Нажмите `Ctrl+H`
3. Замените `C:\Users\...\extracted_xlsx\` на `extracted_xlsx\`

📖 **Подробная инструкция:** см. `EXTRACT_IMAGES_GUIDE.md`

## 🎯 Пошаговая инструкция импорта

### 1️⃣ Загрузка файла
1. Откройте **Прайс (Заготовки)** → кнопка **«Импорт Excel»**
2. Перетащите `.xlsx` файл в зону загрузки или кликните для выбора
3. Выберите целевую категорию из выпадающего списка

### 2️⃣ Настройка маппинга (сопоставление полей)

**Автоопределение:**
Система автоматически распознаёт колонки по названиям:
- `артикул`, `sku`, `article` → **АРТИКУЛ**
- `наименование`, `название`, `title` → **НАИМЕНОВАНИЕ**
- `цена`, `price`, `стоимость` → **ЦЕНА**
- `остаток`, `склад`, `qty` → **ОСТАТОК**
- `photo_filename`, `имя файла` → **ФОТО — имя файла**
- `photo_path`, `путь к фото` → **ФОТО — папка**

**Ручная настройка:**
Если автоопределение сработало неверно:
1. В столбце **«Назначить поле»** выберите нужное поле
2. Красная линия слева показывает назначенные колонки

### 3️⃣ Проверка данных
1. Нажмите **«Далее»** → откроется превью
2. Проверьте корректность маппинга
3. Убедитесь что артикулы распознались верно

### 4️⃣ Запуск импорта
1. Нажмите **«Далее»** → начнётся импорт
2. Прогресс-бар показывает процент выполнения
3. Лог показывает обработанные строки

## ⚙️ Продвинутые возможности

### Динамические поля
Для категорий с дополнительными характеристиками:
1. В маппинге выберите **«✏ Своё поле...»**
2. Введите название характеристики (например: `Диаметр`, `Длина`)
3. Система создаст поле автоматически

### Массовый импорт в несколько категорий
1. Разбейте Excel на несколько файлов по категориям
2. Импортируйте каждый файл отдельно
3. Выбирайте соответствующую целевую категорию

## 🔧 Обработка ошибок

| Ошибка | Причина | Решение |
|--------|---------|---------|
| «Не назначена колонка АРТИКУЛ» | Не выбрано поле для артикула | Вернитесь на шаг 2, назначьте колонку как «АРТИКУЛ» |
| «Артикул уже занят» | Дубликат в базе | Используйте поиск дубликатов или измените артикул |
| Фото не отображается | Неверный путь | Проверьте что папка `extracted_xlsx` существует и путь указан верно |
| «Данные не найдены» | Пустые строки в Excel | Удалите пустые строки или настройте фильтр |

## 💡 Советы

1. **Перед импортом** сделайте бэкап: **Система** → **Локальная копия (.json)**
2. **Тестируйте** на небольшой выборке (10-20 строк)
3. **Проверяйте** превью фото на шаге маппинга
4. **Используйте** префиксы для артикулов чтобы избежать дублей

## 📞 Поддержка

При проблемах с импортом:
1. Проверьте консоль браузера (F12) на ошибки
2. Убедитесь что файл `.xlsx` не повреждён
3. Проверьте что папка `extracted_xlsx` доступна
