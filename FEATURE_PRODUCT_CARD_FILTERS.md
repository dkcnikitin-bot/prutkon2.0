# Карточка товара и Умные фильтры

**Версия:** 18.25.4  
**Дата:** 2024-11-01

---

## 🎯 Обновления

### 1. Улучшенная карточка товара

#### **Что изменилось:**
- ✅ Поля загружаются **автоматически** из схемы категории
- ✅ Иконки для каждого типа поля (⚙️ Тип, 📏 Длина, 🧱 Материал и т.д.)
- ✅ Обязательные поля отмечены флажком
- ✅ Предпросмотр фото 80×80px с fallback
- ✅ Кнопка **Удалить** прямо в карточке
- ✅ Подробный лог изменений в истории
- ✅ Динамическая подгрузка полей при смене категории

#### **Как использовать:**
1. Дважды кликните на товар в таблице → откроется карточка
2. Выберите категорию → поля автоматически обновятся
3. Заполните обязательные поля (с красным флагом)
4. Нажмите **Сохранить** или **Удалить**

---

### 2. Умные фильтры

#### **Доступные фильтры:**

| Фильтр | Описание | Пример |
|--------|----------|--------|
| **Цена от** | Минимальная цена | `100` |
| **Цена до** | Максимальная цена | `5000` |
| **Остаток** | Статус наличия | В наличии / Мало / Нет в наличии |
| **Только с фото** | Есть изображение | ☑️ Включено |
| **Поле категории** | Динамический фильтр | `Техника: Пильный станок` |

#### **Особенности:**
- 🔥 **Динамические поля** — выпадашка заполняется из схемы текущей категории
- ⚡ **Живой поиск** — результаты обновляются при вводе
- 🔄 **Кнопка сброса** — очищает все фильтры одним кликом
- 📊 **Счетчик** — показывает количество активных фильтров

#### **Примеры использования:**

**1. Найти станки дешевле 50000₽ в наличии:**
```
Цена от: 0
Цена до: 50000
Остаток: В наличии
Поле категории: tech_type = Пильный станок
```

**2. Найти товары с фото и остатком < 10:**
```
Остаток: Мало (< 10)
Только с фото: ✓
```

**3. Поиск по материалу:**
```
Поле категории: Материал
Значение: Сталь
```

## 🛠 Технические детали

### Карточка товара

**Функции:**
- `editProduct(id)` — открыть карточку
- `loadProductCategoryFields()` — загрузить поля из схемы
- `saveProductCard()` — сохранить с валидацией
- `deleteProductFromCard()` — удалить с подтверждением

**Валидация:**
- Проверка дубликатов артикула
- Обязательные поля из `requiredFields` категории
- Автолог изменений в `history`

### Фильтры

**Функции:**
- `renderPriceTable()` — основная логика фильтрации
- `clearFilters()` — сброс всех фильтров

**Логика фильтрации:**
```javascript
// 1. Категория (включая подкатегории)
// 2. Текстовый поиск
// 3. Цена (min/max)
// 4. Остаток (in_stock/low_stock/out_of_stock)
// 5. Фото (есть/нет)
// 6. Динамические поля категории
```

## ✅ Проверка

**Как протестировать:**

1. **Карточка товара:**
   - Откройте прайс-лист
   - Дважды кликните на любой товар
   - Проверьте что поля соответствуют категории
   - Смените категорию → убедитесь что поля обновились
   - Измените данные → сохраните → проверьте историю

2. **Фильтры:**
   - Выберите категорию с динамическими полями
   - Убедитесь что выпадашка "Поле категории" заполнилась
   - Примените несколько фильтров
   - Проверьте что таблица отфильтровалась
   - Нажмите "Сброс" → убедитесь что все очистилось

---

## 📝 Примечания

- Динамические поля фильтруются **регистронезависимо**
- Фильтр "Мало" показывает товары с остатком **1-9 штук**
- Если категория не выбрана, фильтры показывают общие значения
- Фото подгружаются из `extracted_xlsx/` (относительный путь)
