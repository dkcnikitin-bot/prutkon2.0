# Политика безопасности ПРУТКОН ОС

## Поддерживаемые версии

В настоящее время обновления безопасности предоставляются только для текущей версии ОС.

| Версия | Поддержка |
| ------- | --------- |
| v10.7.x | ✅ Активна |
| v9.x    | ❌ Прекращена |
| < v9.0  | ❌ Прекращена |

## Сообщение об уязвимости

Мы серьезно относимся к безопасности данных нашего производства. Если вы обнаружили 
уязвимость, пожалуйста, не публикуйте ее открыто.

Пожалуйста, сообщите об обнаруженной проблеме напрямую администратору системы или 
через защищенный канал связи ПРУТКОН ИНДАСТРИАЛ. Мы постараемся ответить в течение 
24 часов и предложить план устранения.

## Меры предосторожности
* Никогда не передавайте свои учетные данные (логин/пароль) третьим лицам.
* При работе в общественных местах всегда завершайте сеанс кнопкой «Выход».
* Не вносите изменения в `core.js` без предварительного тестирования в песочнице.
