# 🛰 Интеграция ПРУТКОН ОС ↔ Bitrix24 (v11.5.0+)

## Архитектура "Fail-Safe Bridge"
Система использует многоуровневую стратегию синхронизации для 100% надежности данных:

1.  **Бинарный транспорт:** Использование Form-Data (GET/POST) вместо JSON для максимальной совместимости.
2.  **Умный Fallback истории:** Авто-переключение между источниками (Activities ➝ Tasks ➝ Timeline), если основные методы заблокированы тарифом.
3.  **Синхронизация состава товаров:** Команда `crm.deal.productrows.set` вызывается одновременно с обновлением суммы, предотвращая конфликты в CRM.

## Шаг 1: Получение входящего вебхука в Bitrix24
Для работы ПРУТКОН ОС нам понадобится специальный «ключ доступа» (webhook) от вашего Битрикса.

1. Зайдите в свой Bitrix24 под аккаунтом администратора.
2. В левом меню выберите пункт **«Приложения»** (или «Маркетплейс»).
3. Перейдите во вкладку **«Разработчикам»** ➝ **«Другое»** ➝ **«Входящий вебхук»**.
4. В настройках вебхука:
   - В названии напишите: **«ПРУТКОН ERP Интеграция»**.
   - В блоке **«Настройка прав»** обязательно выберите:
     - `CRM` (бизнес-логика сделок)
     - `Справочник` (товары)
     - `Пользователи` (для определения ответственного)
5. Нажмите **«Сохранить»**.
6. Вы получите **«Вебхук для вызова REST API»**. Он выглядит примерно так:
   `https://ВАШ_ДОМЕН.bitrix24.ru/rest/1/abcxyz123abc/`
   **Скопируйте его целиком!**

## Шаг 2: Настройка в ПРУТКОН ОС
1. Откройте ПРУТКОН ОС и перейдите в раздел **Сервис ➝ Права доступа и роли** (или Настройки).
2. Найдите новый блок **«Интеграция с Bitrix24»**.
3. Вставьте скопированную ссылку в поле **«Webhook URL»**.
4. Нажмите **«Проверить связь»**. Если все ок — вы увидите уведомление «Связь с Bitrix24 установлена!».

## Шаг 3: Как пользоваться
После настройки в интерфейсе появятся новые возможности:

### А. Из Калькулятора:
При завершении расчета появится кнопка **«Создать Сделку в Bitrix24»**. Система автоматически передаст в Битрикс:
- Название изделия.
- Сумму (из калькулятора).
- Детальную спецификацию в комментарий к сделке.

### Б. Из Менеджера Заказов:
У каждого активного заказа появится иконка **«Отправить в CRM»**. Это обновит статус сделки в Битриксе на «Производство начато» и прикрепит текущие параметры заказа.

*Памятка: Безопасность*
*Ваш Webhook URL является секретным ключом. Не передавайте его третьим лицам, так как он дает доступ к вашей CRM системе.*
